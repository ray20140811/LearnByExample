# Project

## Step 1: Create `main.c` `myfunction.c` `mylib.h` 3 files.

## Step 2: Create the project

```
C-x d
M-x global-ede-mode
M-x ede-new RET Make RET myproject RET
```

## Step 3: Create targets

```
C-c . t myprogram RET program RET
. a myprogram RET
```

## Step 4: Compile

```
C-c . C
```
